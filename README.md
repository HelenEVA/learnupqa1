Возьмём наш текущий проект. Теперь мы нашему классу можем в параметре метода
передавать информацию о скорости не одного человека, а нескольких сразу, упаковав все
скорости в массив. Создайте в этом классе три метода, каждый из которых принимал бы
массив скоростей и возвращал бы:
i) первый метод - количество выбывающих;
ii) второй метод - массив из скоростей выбывающих (в том же порядке, что и в исходном
массива)
iii) третий метод - массив из скоростей не выбывающих (в том же порядке, что и в
исходном массива).
Для второго и третьего методов сперва посчитайте размер ответа, затем создайте массив
ответа и заполните его.
Не забудьте написать тесты на эти методы.
Мавен-координаты библиотеки для идеи: org.junit.jupiter:junit-jupiter:5.4.2
